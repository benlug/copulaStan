# Create a copula_fit object

Create a copula_fit object

## Usage

``` r
copula_fit(fit, copula, marginals, data_dim)
```

## Arguments

- fit:

  CmdStanMCMC fit object.

- copula:

  Copula type string.

- marginals:

  Character vector of marginal distribution names.

- data_dim:

  Dimensions of the input data.

## Value

A `copula_fit` S3 object.
